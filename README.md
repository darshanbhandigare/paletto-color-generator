# 🎨 Paletto — Color Palette Generator

A modern, interactive color palette generator that helps designers and developers create beautiful color combinations instantly.

🚀 Built with pure HTML, CSS, and JavaScript — no frameworks needed.

---

## ✨ Features

- 🎯 Generate palettes using multiple harmony modes:
  - Complementary
  - Analogous
  - Triadic
  - Split Complementary
  - Tetradic
  - Monochromatic

- 🎨 Real-time color preview (Strip & Card views)

- 🌈 Tints & Shades generator for every palette

- 📋 One-click copy for HEX values

- 💾 Save palettes locally (LocalStorage)

- 📦 Export palettes in multiple formats:
  - CSS Variables
  - HEX Array (JS)
  - Tailwind Config
  - SCSS Variables

- 🔄 Random color generator

## ⚙️ How It Works

- Converts HEX → HSL for better color manipulation  
- Applies color harmony algorithms  
- Generates palette variations dynamically  
- Renders UI in real-time using DOM manipulation
